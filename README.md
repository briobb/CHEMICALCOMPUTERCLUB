# Codex Instructions  
## Chemical Computer Club Official Website

### 目的

**Chemical Computer Club（CCC）**の公式ブランドサイトを制作してください。

CCCは、1986年に設立された研究クラブという設定を持つアパレル・ライフスタイルブランドです。

ブランドの目的は、

**化学とコンピューターの力で、暮らしをより豊かにすること**

です。

世界観はありますが、暗い秘密結社や怪しい研究施設のようには見せないでください。

CCCは、好奇心、発見、ものづくり、自然の中で遊ぶことを楽しむ、明るく前向きなクラブです。

---

# ブランド設定

Chemical Computer Clubは1986年に設立された研究クラブです。

化学、コンピューター、ものづくり、自然観察、アウトドア活動などを通して、日々の暮らしをより面白く、豊かにすることを目的としています。

現在もクラブは活動を続けています。

CCCの商品は、一般的なブランドグッズではなく、クラブ員のために制作されたウェアや道具という位置づけです。

商品を購入した人は、Chemical Computer Clubのクラブ員になります。

---

# ターゲット

主なターゲットは、30代から50代の男女です。

以下のような人を想定してください。

- キャンプや登山、釣り、サイクリングが好きな人
- 古いコンピューターや工業製品が好きな人
- ガジェットや工具、ものづくりが好きな人
- グラフィックTシャツやデザイン性の高い服が好きな人
- レトロフューチャーや1980年代のカルチャーが好きな人
- 大人が自然に着られる、少し遊び心のある服を探している人

アウトドアブランドのような健全さと、研究クラブらしい知的好奇心を両立してください。

# カラーパレット

## Archive Ivory

メイン背景色。

```css
#F3F0E8
```

1980年代のコンピューター筐体、研究資料、古い紙を連想させる生成り色です。

サイト全体の基本背景として使用してください。

## Graphite Black

メインテキスト、ロゴ、見出し。

```css
#2B2B2B
```

純粋な黒は避け、印刷インクのような柔らかい黒を使用してください。

## Safety Orange

コーポレートカラー、アクセント。

```css
#D66A2E
```

ボタン、リンク、重要な情報、ラベルなどに使用してください。

使用量は全体の5％から10％程度に抑えてください。

オレンジは危険や不安を表す色ではなく、発見、活動、前向きなエネルギーを表す色として使用してください。

## Steel Gray

補助テキスト、罫線、図面表現。

```css
#8E9497
```

# タイポグラフィ

装飾的な未来フォントは使わず、実在する研究機関や工業製品に使われていそうな書体を使用してください。

推奨書体：

- Helvetica Neue
- Arial
- Inter
- DIN系フォント
- IBM Plex Sans
- IBM Plex Mono

使用ルール：

- 通常の見出しと本文はサンセリフ体
- 製品番号、日付、クラブ員番号、技術情報は等幅フォント
- ロゴはシンプルで読みやすくする
- OCRフォントは補助的にのみ使用する
- デジタル時計風フォントやゲーム風フォントは使わない

Google Fontsを使用する場合は、以下を推奨します。

- Inter
- IBM Plex Sans
- IBM Plex Mono

# UIルール

- 角丸は小さくするか、基本的に使用しない
- ボタンはシンプルな長方形
- 太い影は使用しない
- グラデーションは使用しない
- ガラス表現は使用しない
- 写真をカードの枠で囲みすぎない
- 余白を多く取る
- 線や罫線を効果的に使う
- セクション番号を小さく表示する
- 技術資料のようなラベルを補助的に使用する
- オレンジは重要箇所のみに使用する
- アニメーションは控えめにする

---

# アニメーション

使用してよい表現：

- ゆっくりしたフェードイン
- 少量のスライドアップ
- ホバー時の線や文字色の変化
- 写真のわずかな拡大
- オレンジ色の小さなインジケーター

使用しない表現：

- 激しいパララックス
- グリッチ
- 画面の点滅
- CRTノイズ
- タイピング演出の多用
- 起動音や自動再生音声
- 過剰なローディング画面

---

# レスポンシブ対応

スマートフォンを重視してください。

- 横スクロールを発生させない
- 文字サイズを適切に調整する
- タップ領域を十分に確保する
- 商品画像は縦に見やすく配置する
- 活動内容やJournalは横スライド表示でも構わない
- モバイルでも余白を残す
- ハンバーガーメニューを実装する

---

# 技術要件

- HTML5
- CSS3
- Vanilla JavaScript
- 外部フレームワークは使用しない
- GitHub Pagesで公開できる静的サイト
- `index.html`
- `css/style.css`
- `js/main.js`
- `images/`フォルダ
- セマンティックHTML

- コードには分かりやすいコメントを入れる

# 最終目標

初めてサイトを訪れた人が、最初に感じるのは「怪しい」ではなく、

**明るくて、知的で、格好いいブランド**

であることです。

次に、

**Chemical Computer Clubとは何だろう**

と興味を持ち、ブランドストーリーや活動を読み進めます。

最終的に、

**このクラブのアイテムを持って、クラブ員になりたい**

と感じられるサイトにしてください。

世界観よりも、まずビジュアルの魅力を優先してください。

ストーリーを知らなくても格好いい。

ストーリーを知ると、さらに好きになる。

その順番を大切にしてください。
